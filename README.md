# Resources_for_Robotics


通过 Git 将学习资源同步至你的磁盘。  
顺带学一学 Git.  
欢迎 Pull Request.  

如果你只想下载，可以通过 [**坚果云的链接**](https://www.jianguoyun.com/p/DeVkFX4Q-OOjBxjO_OIC )（速度会快一些）

克隆本仓库：
```
$ git clone git@github.com:linyuxuanlin/Resources_for_Robotics.git
```

获取更新：
```
$ git pull
```
